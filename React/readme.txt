Первое что нужно сделать - создать проект react.
Используем Create React App
Для этого создаем пвпку проекта и прямо в ней без созданных файлов вызываем в терминале:
        npx create-react-app my-app
Далее заходим в папку с приложением в терминале
        cd my-app
Рекомендовано отключить плагин js hint, если он установлен - будет конфликтовать
Далее запускаем
        npm start


Структура в документе index.js означает следующее: 
        import React from 'react';   // импорт библиотеки из нод модулей, отвечающей за препроцессор JSX, методы реакта
        import ReactDOM from 'react-dom/client'; //импорт библиотеки, отвечающей за добавление/взаимодействие элементов реакт с DOM деревом проекта 
        import './index.css'; //стандартный импорт стилей
        import App from './App'; //импорт функции App из файла App.js - это функция конструктор, в которую будут стекаться все части кода из разных файлов
        import reportWebVitals from './reportWebVitals'; // библиотека, измеряющая скорость работы приложения. Сейчас нам не нужна, удаляем и импорт и ее вызов


Настройка Emmet для реакт - 
File/Preferences/Settings/Extensions/Emmet/Include Languages/Add Item
key: javascript
Value: javascriptreact

Теперь вводим тег html, нажимаем Tab, легко и быстро прописываются теги


В реакте вместо html аттрибута (селектора) class Используем className, потому что class уже занято и в реакте продумана альтернатива.
По той же причине вместо аттрибута for - htmlFor - по той же причине
